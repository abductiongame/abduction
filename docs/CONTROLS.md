# Controls

Abduction supports desktop and mobile input. An interactive onboarding sequence appears on first load.

## Desktop

| Action | Input |
|--------|-------|
| Move UFO | **Left-click + drag** |
| Orbit camera | **Right-click + drag** |
| Zoom | **Scroll wheel** |

## Mobile / touch

| Action | Input |
|--------|-------|
| Move UFO | **One finger** drag |
| Rotate camera & zoom | **Two fingers** (pinch / rotate) |

## In-game UI

| Element | Action |
|---------|--------|
| Star icon (header) | Open the About panel |
| About panel close (X) | Close About panel |
| Audio button (footer) | Toggle sound |
| **Documentation** (bottom bar) | Open GitHub docs |
| **@abductiongame** (bottom bar) | Open community on X |

## Tips

- The tractor beam activates automatically when you fly over crowds — no separate button.
- There is no pause menu; refresh the page to reset the simulation.
- Performance scales with GPU — lower-end devices may drop frames with the full crowd active.
